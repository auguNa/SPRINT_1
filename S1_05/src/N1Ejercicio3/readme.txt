To run the FolderLister program from the command line, follow these steps:

    1. Comment the line 2 from Main.java & FolderLister file (//package S1_05.N1Ejercicio3;)
    2. Comment line 24 in Main.java (//String filePath = currentDirectory....)
    3. Enable line 26 & 27 in Main.java (String filePath = currentDirectory  + File.separator + file;
                                                   System.out.println("path "+ filePath);)
    4. Open a new Command Prompt window
    5. Navigate to the root directory of the project: cd 'project_root'
    6. Compile the Java source file: javac Main.java
    7. Run the compiled program java Main
