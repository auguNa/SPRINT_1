To run the FolderLister program from the command line, follow these steps:

    1. Comment the lines  from Main.java & FolderLister file (//S1_05.N1Ejercicio4;)

    2. Open a new Command Prompt window
    3. Navigate to the root directory of the project: cd 'project_root'
    4. Compile the Java source file: javac Main.java
    5. Run the compiled program java Main
    6. Insert the file path you want to read e.g. "C:\S1_05\N1Ejercicio4\readme.txt"
    7. Enjoy the results :)
